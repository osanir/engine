# engine

# Libraries
`libsfml-dev`

# Compile & Run
Compile:
```bash
 g++ main.cpp sources/*.cpp -o game -lsfml-network -lsfml-graphics -lsfml-window -lsfml-system -lGL -lsfml-audio
```
Run:
```bash
./game
```
## or
Give execution permission:.
```bash
chmod +x compile.sh
```
Run compile script:
```bash
./compile.sh
```
